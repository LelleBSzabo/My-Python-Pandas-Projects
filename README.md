# Retail & Sales Data Analysis Project — Data Technician Bootcamp

This project was completed during a Data Technician bootcamp and focuses on exploring, cleaning, and analysing retail and sales datasets using Python. It demonstrates practical use of pandas for data manipulation and matplotlib for data visualisation.

## Project Overview

The project involved:

- Exploring raw retail and sales datasets
- Cleaning and preparing data for analysis
- Filtering and subsetting data to answer specific questions
- Applying aggregation and transformation methods
- Visualising patterns and trends using Python plotting libraries

## Skills and Tools Used

### Data Filtering and Subsetting

Techniques used:

- `df.loc[]` for label-based row and column selection
- `df.iloc[]` for index-based selection
- Boolean filtering to extract targeted subsets of the data

### Data Cleaning and Preprocessing

Pandas functions applied:

- `dropna()` to remove missing values
- `fillna()` to replace missing entries
- `astype()` to correct and standardise data types
- `drop_duplicates()` to remove repeated records

### Data Manipulation and Aggregation

Methods used for analysis:

- `groupby()` to summarise data across categories
- `sort_values()` to rank sales, products, or regions
- `value_counts()` to explore distributions
- Creating new calculated fields such as revenue or totals

### Data Visualisation

Visualisations created using:

- Pandas built-in plotting functions
- Matplotlib for more customised charts

Charts included bar charts, line plots, histograms, and comparisons of sales metrics.

## What I Learned

This project developed skills in:

- Using pandas for structured data analysis
- Cleaning and preparing datasets for modelling or reporting
- Extracting insights through grouping, sorting, and aggregations
- Presenting findings visually using Python libraries
- Performing end-to-end analysis on retail and sales data
