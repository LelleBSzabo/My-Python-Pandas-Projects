# 🛒 Retail & Sales Data Analysis Project

This project was completed as part of a **Data Technician Bootcamp** and focuses on cleaning, analysing, and visualising retail sales data using **Python**.

---

## 📊 Project Overview
- Analysed a retail dataset containing **transactional sales data** across products, regions, and time periods  
- Cleaned and prepared raw data by handling **missing values**, correcting **data types**, and removing **duplicates**  
- Applied **filtering, grouping, and aggregation techniques** to uncover key sales trends  
- Created **visualisations** to communicate insights effectively  

---

## 🛠️ Key Skills & Techniques

### Data Cleaning & Preparation
- 🧹 Handled missing data using `dropna()` and `fillna()`  
- 🛠️ Standardised data types with `astype()`  
- ❌ Removed duplicate records to improve data quality  

### Data Analysis
- 📈 Used `groupby()` to analyse sales by product, region, and time  
- 🔢 Applied `sort_values()` and `value_counts()` to identify top-performing categories  
- ➕ Created calculated fields such as **total revenue**  

### Data Visualisation
- 📊 Built charts using **pandas** (bar charts, line plots, histograms)  
- 🌟 Visualised trends in sales performance and category comparisons  

---

## 🎯 Key Outcomes
- 🏆 Identified high-performing products and regions  
- 📉 Revealed sales trends and distribution patterns across the dataset  
- 💡 Transformed raw data into **meaningful insights**  

---

## 💡 What I Learned
- 🧠 How to clean and prepare **real-world datasets** for analysis  
- 🔍 How to extract **actionable insights** using pandas  
- 🎨 How to effectively communicate findings through **data visualisation**  
- 🚀 Gained experience in **end-to-end data analysis workflow** from raw data to insights
